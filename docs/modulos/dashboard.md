# Dashboard

El Dashboard es el panel principal de GMAS que proporciona una vista general del sistema con información relevante y métricas clave.

## Características Principales

- **Vista General**: Visualización rápida de estadísticas importantes
- **Acceso Rápido**: Navegación directa a los módulos principales del sistema
- **Información en Tiempo Real**: Datos actualizados del estado del sistema

## Interfaz de Usuario

![Dashboard General](/img/dashboard/dashboard-general.png)

El dashboard presenta:

- Métricas resumidas del sistema
- Acceso directo a módulos principales
- Información relevante para la gestión diaria

## Funcionalidades

### Visualización de Datos

El dashboard muestra información consolidada que permite:

- Monitorear el estado general del sistema
- Identificar métricas clave de un vistazo
- Acceder rápidamente a las funciones más utilizadas

### Navegación

Desde el dashboard puedes acceder a:

- Gestión de Compañías
- Administración de Usuarios
- Gestión de Vendedores
- Configuración del Sistema

## Notas Importantes

:::tip Personalización
El dashboard se adapta según los permisos del usuario, mostrando únicamente la información y módulos a los que tiene acceso.
:::

:::info Actualización
Los datos del dashboard se actualizan automáticamente para reflejar el estado actual del sistema.
:::
